# Cadastramento de Currículos

Aplicação web responsiva para cadastro e organização de currículos. O projeto permite registrar informações pessoais, experiências profissionais, formações e competências, armazenando os dados localmente no navegador.

## Funcionalidades

- Formulário completo dividido por seções para facilitar o preenchimento dos dados.
- Campos dinâmicos para adicionar múltiplas experiências profissionais e formações acadêmicas.
- Lista de currículos cadastrados com resumo das informações e opção de exclusão.
- Armazenamento local usando `localStorage`, permitindo revisar os cadastros mesmo após recarregar a página.
- Interface moderna, responsiva e em português.

## Como executar

Basta abrir o arquivo `index.html` em um navegador moderno. Para utilizar um servidor local simples, execute:

```bash
python -m http.server
```

Em seguida acesse `http://localhost:8000`.
